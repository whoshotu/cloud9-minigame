# Strike Force Targets

A fast-paced Duck Hunt-style arcade game for Cloud9 x JetBrains Hackathon - Event Mini-Game category.

## About

Strike Force Targets is an arcade clicking game where players shoot League of Legends and Valorant champions as they pop up on screen. Built for esports event booths with quick 90-second gameplay sessions and competitive high score tracking.

## How to Play

1. Open `index.html` in any modern web browser
2. Toggle audio on/off if desired
3. Click "START GAME" 
4. Watch for the "3, 2, 1, GO!" countdown
5. Click on champion targets before they escape off screen
6. Build combo streaks for bonus points
7. Don't miss 3 targets or it's game over!

## Game Mechanics

- **Timer:** 90 seconds per game
- **Lives:** 3 hearts (lose 1 when target escapes)
- **Scoring:** +10 points per hit, combo multipliers for streaks
- **Targets:** League and Valorant champions with varying speeds
- **Bonus Target:** C9 logo worth 50 points
